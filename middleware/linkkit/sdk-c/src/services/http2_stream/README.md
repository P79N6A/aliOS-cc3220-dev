## Introduction

### Features
- h2 stream upload
- h2 stream download
- file upload over http2 stream

### Dependencies
- **hal**. osal and hal to shield different os and hardware
- **infra**. Authentication, net and so on tool set.
- **http2**. http2 protocpl
